# 动物世界-野生动物识别系统

- 项目简介
  - 本产品主要面向野生动物爱好者，满足该类人群野外探险时打卡各类野生动物的需求。利用计算机视觉，机器学习算法根据动物图像，进行森林动物物种识别和分类，完成一个能够准确识别不同森林动物物种的模型，有助于促进野生动物知识的普及和野生动物的保护。
